About this site

Purpose

This is an experimentation website used to explore common web building blocks: multiple pages, navigation, headings, image galleries, animated GIFs, video players and an interactive slider.

Tech

It is a static site built with plain HTML, CSS and JavaScript, and is designed to be hosted on GitHub Pages. Adobe Target's at.js is integrated on every page to support experimentation and personalization.

Pages

  • Home – landing page and overview.
  • Images & GIFs – responsive gallery.
  • Video – HTML5 and embedded video.
  • Slider – auto-playing carousel.

Built as an experimentation project · Hosted on GitHub Pages
